THE GREAT
RGB (r, g, b)
COLOUR GAME

Points !

  • Every colour in the Colour Palette has a unique RGB value
  • A RGB colour code is given at the top at each play
  • The Player has to select a Colour square corresponding the given colour
  • A maximum of 6 tries will be given at each game play
  • A score of +5 will be given for each correct answer and -1 for each wrong ones
  • The game stops when a player runs out of tries
  • You can switch between the easy mode and the hard mode of the game play
  • A Player is allowed to switch between Easy and Hard mode anytime
  • Switching modes will Reset the game!
  • A player can choose the new sets of colours any time
  • Choosing New colours would not reset the game
